Already Done:


Java - Java is a high-level, object-oriented programming language developed by Sun Microsystems (now owned by Oracle). It is known for its "Write Once, Run Anywhere" capability, as Java programs run on the Java Virtual Machine (JVM), making them platform- independent. Java is widely used for web development, mobile apps (Android), enterprise applications, and game development. It supports features like automatic memory management (garbage collection), multithreading, and a rich standard library. Popular Java frameworks include Spring, Hibernate, and JavaFX.



HTML - HTML (HyperText Markup Language) is the standard language used to create and structure web pages. It uses tags to define elements like headings, paragraphs, links, images, and tables. HTML works alongside CSS (for styling) and JavaScript (for interactivity) to build modern websites. It follows a hierarchical structure, with elements nested inside one another. The latest version, HTML5, introduces new semantic tags, multimedia support, and improved accessibility.

Ongoing:


CSS - CSS (Cascading Style Sheets) is a stylesheet language used to control the appearance and layout of HTML elements. It allows developers to apply styles such as colors, fonts, spacing, and animations to web pages. CSS follows a cascade system, meaning styles can be applied at different levels (inline, internal, or external). The latest version, CSS3, introduces features like flexbox, grid, transitions, and animations, making modern web design more responsive and visually appealing.

Planning To Do:

JavaScript is a powerful and versatile programming language used to create interactive and dynamic web applications. It runs both in web browsers (client-side) and on servers using Node.js, making it essential for full-stack development. JavaScript enables features like real-time updates, animations, and API interactions, enhancing user experience. It is a core web technology alongside HTML and CSS, and its vast ecosystem includes popular frameworks like React.js, Angular, and Vue.js. Widely used in modern web development, JavaScript plays a crucial role in building responsive and interactive websites and applications.
